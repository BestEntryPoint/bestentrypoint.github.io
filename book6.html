 So now we're going to build probably the simplest Neural Network that would help you make the decision of whether you go at the party, or you stay home.

So, for example, if we know that at the party there is your best friend, there isn't your favorite cocktail and it isn’t raining then we have the following:




In programming, such facts converted into numbers — either 0 or 1, for example — are called Inputs:


Next, the machine has to multiply each Input by an individual Multiplier:


Multiplier 1 =  0.5
Multiplier 2 =  0.5
Multiplier 3 = -0.5

1 * 0.5     = 0.5
0 * 0.5     = 0.0
0 * (-0.5) = 0.0

These Multipliers (0.5, 0.5, -0.5) are very important in building the AI. Finding such Multipliers is one of the most important tasks while building any AI.

Anyway, for educational purposes, let's suppose that we have already been given those Multipliers:
Multiplier 1 =  0.5
Multiplier 2 =  0.5
Multiplier 3 = -0.5

Okay.

Next, as we can see below, after the machine multiplies our Inputs with their specific Multipliers respectively, the machine sums the results:


1 * 0.5     = 0.5
0 * 0.5     = 0.0
0 * (-0.5) = 0.0
---------------------
sum:          0.5


So, sum = 0.5.

Next, the machine puts this sum into the main formula looking this way:


  if sum >= 0.5  ——  I'd like to recommend you to GO to the party, my host
if sum <   0.5  ——  I'd like to recommend you to STAY home, my host
  


As we can see, everything is easy!

If our sum is equal to or more than 0.5, then the machine says —— I'd like to recommend you to GO to the party, my host.

But else (for example, if our sum is equal to 0), then the machine says —— I'd like to recommend you to STAY home, my host.

Since our sum is equal to 0.5 and, it means, the sum meets the condition that If our sum is equal to or more than 0.5, the machine will say: I'd like to recommend you to GO to the party, my host.


Of course, when there are other Inputs then the recommendation may be completely other.

It is exactly the way how any AI works.

For now, you need to understand just some things.

The 1st thing
There are some Inputs — some facts of life that may be converted (turned) into numbers — 1 or 0, for example.

The 2nd thing
There are some Multipliers to multiply the Inputs by.

The 3rd thing
There is some main Formula by which the machine makes its recommendation.

As we can see, the machine loves to work with numbers.

For this reason, let's agree to consider the recommendation to GO as 1 and the recommendation to STAY as 0.
